  <action>
    Add `pub mod events;` to `crates/parser-core/src/lib.rs`.

    Create `crates/parser-core/src/events.rs` with:

    ```rust
    pub fn normalize_combat_events(
        raw: &RawReplay<'_>,
        entities: &[ObservedEntity],
        context: &SourceContext,
        diagnostics: &mut DiagnosticAccumulator,
    ) -> Vec<NormalizedEvent>
    ```

    Required behavior:

    - Build a `BTreeMap<i64, &ObservedEntity>` from normalized entities.
    - Iterate `raw::killed_events(*raw)` in source order.
    - Event IDs must be exactly `event.killed.{event_index}`.
    - Source refs must include `context.event_source_ref(...)` with rule ID matching the semantic rule.
    - Actor refs must use observed entity ID, observed name, and side from `ObservedEntity`.
    - Null killer + player victim:
      - `kind: NormalizedEventKind::Death`
      - `combat.semantic: NullKillerDeath`
      - killer is `FieldPresence::ExplicitNull { reason: NullReason::NullKiller, ... }`
      - bounty exclusion contains `NullKiller`
      - legacy effects include victim `isDead` delta only
    - Enemy player kill:
      - `kind: NormalizedEventKind::Kill`
      - `combat.semantic: EnemyKill`
      - bounty state `Eligible`
      - legacy effects include killer `kills +1`, victim `isDead +1`, relationship effects for `killed` and `killers`
    - Same-side non-suicide player kill:
      - `kind: NormalizedEventKind::Teamkill`
      - `combat.semantic: Teamkill`
      - bounty exclusion contains `Teamkill`
      - legacy effects include killer `teamkills +1`, victim `isDead +1`, victim `isDeadByTeamkill +1`, relationship effects for `teamkilled` and `teamkillers`
    - Suicide player kill:
      - `kind: NormalizedEventKind::Suicide`
      - `combat.semantic: Suicide`
      - bounty exclusion contains `Suicide`
      - legacy effects include victim `isDead +1` only
    - Player killer + vehicle/static victim:
      - `kind: NormalizedEventKind::VehicleKilled`
      - `combat.semantic: VehicleDestroyed`
      - bounty exclusion contains `VehicleVictim`
      - legacy effects include killer `vehicleKills +1`
    - Unknown/malformed actor or victim:
      - `kind: NormalizedEventKind::Unknown`
      - `combat.semantic: Unknown`
      - bounty exclusion contains `UnknownActor`
      - push diagnostic code `"event.killed_actor_unknown"` or `"event.killed_shape_unknown"` with `DiagnosticImpact::DataLoss`
      - legacy effects are empty

    Use rule IDs:

    - `event.killed.enemy`
    - `event.killed.teamkill`
    - `event.killed.suicide`
    - `event.killed.null_killer`
    - `event.killed.vehicle_destroyed`
    - `event.killed.unknown`
  </action>
