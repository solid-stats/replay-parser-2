## Sampling Rate

- **After every task commit:** Run the narrow package tests named in that task's `<verify>` block.

- **Before `$gsd-verify-work`:** Full suite must be green, and Phase 5.2 benchmark acceptance evidence must pass or record a blocking gap.
- **Max feedback latency:** 900 seconds for code waves, excluding explicit all-raw benchmark runs.
