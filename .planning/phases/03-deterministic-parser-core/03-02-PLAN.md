    Implement helpers:

    ```rust
    impl<'a> RawReplay<'a> {
        pub const fn new(root: &'a serde_json::Map<String, serde_json::Value>) -> Self;
        pub fn value_at(&self, key: &str) -> Option<&'a serde_json::Value>;
        pub fn string_field(&self, key: &str) -> RawField<'a, String>;
        pub fn u64_field(&self, key: &str) -> RawField<'a, u64>;
        pub fn f64_field(&self, key: &str) -> RawField<'a, f64>;
        pub fn u32_vec_field(&self, key: &str) -> RawField<'a, Vec<u32>>;
        pub fn array_field(&self, key: &str) -> RawField<'a, &'a Vec<serde_json::Value>>;
    }
    ```

    Define a documented `RawField<T>` enum with variants:

    ```rust
    Present { value: T, json_path: String },
    Absent { json_path: String },
    Drift { json_path: String, expected_shape: &'static str, observed_shape: String },
    ```

    Implement `observed_shape(&serde_json::Value) -> String` returning stable strings such as `string`, `number`, `array`, `object`, `null`, and `boolean`.

    Update `artifact.rs` so successful root objects construct `RawReplay` and pass it to metadata normalization in Task 2. Do not normalize entities yet.

    Populate `ReplayMetadata` exactly:

    - `missionName` -> `mission_name`

    - `endFrame` -> `end_frame`

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| raw top-level JSON -> metadata contract | Source fields can drift or be absent; output must remain explicit and auditable. |
| metadata artifact -> server-2 persistence | Later backend consumers need stable field names and presence semantics. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-02-01 | Tampering | metadata field extraction | mitigate | Preserve source refs and exact JSON paths for every present or drifted field. |
| T-03-02-02 | Repudiation | schema drift diagnostics | mitigate | Emit stable diagnostic codes with expected/observed shape and parser action. |
| T-03-02-03 | Denial of Service | malformed local fields | mitigate | Treat localized drift as unknown/diagnostic instead of panicking. |
</threat_model>
