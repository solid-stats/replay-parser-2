## Deviations from Plan

### Resolved Decision

**1. [Tooling] Stable Rust could not support narrow native coverage exclusions**

- **Found during:** Task 3 strict coverage verification
- **Issue:** Native `cargo llvm-cov` thresholds could report gaps, but stable Rust could not subtract exact source-line exclusions. `#[coverage(off)]` is experimental on rustc 1.95.0, and `--ignore-filename-regex` would create unacceptable whole-file exclusions.
- **Decision:** Use a custom `cargo llvm-cov --json` postprocessor instead of weakening the allowlist policy.
- **Verification:** `scripts/coverage-gate.sh` now fails on unallowlisted production uncovered regions and passes with the committed exact-line allowlist.
